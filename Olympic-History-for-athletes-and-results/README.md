# Olympic History Data Analysis (1896-2016)
Comprehensive analysis of Olympic Games data from 1896 to 2016. Exploring athlete statistics, medal distributions, country performance, and historical trends using Python and data visualization.

## 📊 Project Overview
Comprehensive analysis of 120 years of Olympic history data exploring athlete demographics, medal trends, country performance, and historical insights.

## 🛠 Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## 📁 Dataset
Analysis of Olympic games from Athens 1896 to Rio 2016, including athlete information, events, and medal results.

## 🚀 How to Run
1. Clone this repository: `git clone https://github.com/yourusername/Olympic-History-for-athletes-and-results.git`
2. Install required packages: `pip install pandas numpy matplotlib seaborn jupyter`
3. Open the Jupyter notebook: `jupyter notebook olympic-analysis.ipynb`
